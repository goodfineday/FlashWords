# FlashWords - 快速记忆英语单词词库与配图项目

## 项目介绍

FlashWords 是一个专注于 **高效英语单词记忆** 的开源项目，提供结构化的单词词库和标准化记忆配图。核心目标是通过「科学记忆方法 + 可视化配图 + 完整单词信息」的组合，帮助学习者快速建立单词认知、强化记忆效果，同时支持开发者二次集成或扩展使用。

项目核心特色：



* 单词信息全覆盖：从音标、音节到词源、短文，满足不同学习场景需求

* 科学记忆体系：内置对比记忆、场景记忆、联想记忆等多种速记方案

* 标准化配图：16:10 分辨率配图，精准匹配单词记忆逻辑（如联想场景、反义词对比等）

* 易扩展格式：采用 JSON 结构化存储，支持跨平台导入、二次开发

## 核心功能



| 功能模块        | 详细说明                                                       |
| ----------- | ---------------------------------------------------------- |
| 完整单词数据      | 包含音标（英 / 美）、音节划分、多词性词义、短语、例句、同反义词、词源等 12 + 维度              |
| 多元速记方案      | 每个单词配套对比记忆、场景记忆、联想记忆等方法，针对性突破记忆难点                          |
| 标准化记忆配图     | 16:10 分辨率图片，视觉化呈现单词核心含义 / 记忆逻辑（如 `above` 配图可展示「窗户上方的鸟窝」场景） |
| 结构化 JSON 格式 | 数据易解析、易修改，支持导入 Anki、自定义 App 等第三方工具                         |
| 实用短文拓展      | 每个单词配套主题短文，强化语境记忆，衔接单词用法                                   |

## 单词数据格式说明

所有单词以 JSON 格式存储，单条数据结构如下（以 `above` 为例）：



```
{

&#x20; "word": "单词拼写",

&#x20; "content": "### 音标 \n\n 英：/xxx/；美：/xxx/。\n\n### 音节 \n\n 音节数量、划分方式、重音位置。\n\n### 词义 \n\n 多词性说明+核心含义+适用场景。\n\n### 短语 \n\n 1. 短语1 - 含义（场景标注）\n 2. 短语2 - 含义（场景标注）\n\n### 例句 \n\n 1. 英文例句1 - 中文翻译\n 2. 英文例句2 - 中文翻译\n\n### 同反义词 \n\n 同义词：\n 1. 单词/短语1（词性）：说明+替换例句\n 反义词：\n 1. 单词/短语1（词性）：说明+对比例句\n\n### 同根词 \n\n 同根词列表+词性+含义（无则标注“无明显同根词”）\n\n### 词根词缀 \n\n 词缀拆分+词源演变说明（无则标注对应说明）\n\n### 词源 \n\n 词汇历史演变+含义扩展历程\n\n### 变形 \n\n 单复数、时态等变形（无则标注“无变形”）\n\n### 速记 \n\n 对比记忆+场景记忆+联想记忆等多种方法\n\n### 短文 \n\n 包含该单词的主题短文（中英文对照）"

}
```

### `content` 核心模块说明



| 模块         | 作用              |
| ---------- | --------------- |
| 音标 / 音节    | 解决发音痛点          |
| 词义 + 适用场景  | 明确使用边界          |
| 短语 + 例句    | 强化搭配记忆          |
| 同反义词 / 同根词 | 建立词汇网络          |
| 词源 / 词根词缀  | 理解词汇本质（适用于深度学习） |
| 速记         | 提供高效记忆技巧        |
| 短文         | 语境化应用           |

## 配图规范



* 分辨率：严格遵循 16:10 比例（如 1920×1200、1280×800 等）

* 内容要求：

1. 紧扣单词核心含义或速记逻辑（如 `above` 配图需体现 “上方” 位置关系）

2. 画面简洁、无冗余元素，突出记忆关键点

3. 优先使用实景图 / 示意图，避免抽象难懂的设计

* 命名规则：与单词拼写一致（如 `above.png`），存储于 `assets/images/` 目录

## 使用指南

### 1. 直接使用词库



* 下载 `words/` 目录下的 JSON 文件，可直接导入支持 JSON 格式的记忆工具（如 Anki、Memrise 等）

* 查看 `assets/images/` 目录获取对应单词配图，搭配词库使用

### 2. 二次开发集成



* 读取 JSON 文件解析单词数据，通过 `word` 字段关联配图文件

* 支持扩展功能：如按词性 / 难度筛选单词、随机生成单词卡片等

### 3. 本地预览



1. 克隆仓库：`git clone ``https://github.com/your-username/FlashWords.git`

2. 直接打开 `words/` 目录下的 JSON 文件查看单词数据

3. 访问 `assets/images/` 查看配图

## 贡献指南

欢迎通过以下方式参与项目贡献：

### 1. 提交新单词



* 按照项目 JSON 格式填写完整单词信息（确保 `content` 模块无缺失）

* 配套提供 16:10 分辨率配图，命名与单词一致

* 提交 PR 至 `words/` 目录（新增文件命名：`单词.json`）

### 2. 优化现有内容



* 修正音标、例句等错误信息

* 补充更优的速记方法或短文

* 替换质量不佳的配图（需符合规范）

### 3. 贡献规范



* 单词信息需准确（优先参考牛津 / 朗文词典）

* 配图需无版权问题（建议使用 Pixabay、Pexels 等免费图库）

* 不提交重复单词，提交前请搜索项目现有词库
